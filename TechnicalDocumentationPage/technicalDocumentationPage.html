Introduction

JavaScript is a high-level, interpreted programming language.

It is used to add interactivity to websites and web apps.

Originally developed for the browser, it is now used on servers too.

What you should already know

Before diving into JavaScript, you should understand the following basics:

  • Basic HTML structure
  • CSS syntax and usage
  • What a browser does
  • Using the browser's Developer Tools
  • Basic understanding of programming logic

This knowledge will help you understand how JS fits into web development.

JavaScript and Java

Despite the similar names, JavaScript and Java are distinct languages.

Java is a strongly typed, compiled language used for large applications.

JavaScript is dynamically typed and typically used for client-side scripting.

Hello world

Let's write your first JavaScript code!

This simple example prints a message in the browser's console:

console.log('Hello, World!');

Open your browser console (F12 or Ctrl+Shift+I) to see the output.

Variables

Variables are containers for storing data values.

You can declare variables using var, let, or const.

Example using let:

let userName = "FreeCodeCamp";

Use const when the value should not change:

const pi = 3.14159;
